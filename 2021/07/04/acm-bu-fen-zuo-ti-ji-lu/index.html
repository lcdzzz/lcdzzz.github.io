hdoj 2036改革春风吹满地

  • 简述：（变成一般问题的话）一个多边形告诉你边数（定点数）为n，然后是按照逆时针顺序给出的n个顶点的坐标（x1, y1, x2, y2… xn, yn），然后求他的面积。

这题最简单最快的方法就是叉乘法（原理不懂不要紧，记住公式，反正很容易记）

部分代码实现

for (int i = 0; i < n; i++) {
	                int t = i + 1;
  	              if (t == n) {
     	               t = 0;
    	            }
    	            s = s+(a[i] * b[t] - a[t] * b[i])/2.0;
      	      }

这当中，数组a代表x坐标，数组b代表y坐标，从i=0（第一个点）叉乘到i=n-1（最后一个点）

这些点是逆时针顺序的，如果是顺时针，那么求出来的值是负数，就要在前面加一个绝对值Math.abs

上面代码翻译成数学就是:

s = [（x1 * y2 - x2 * y1）+（x2 * y3 - x3 * y2) + ….. + (x(n - 2) * y(n - 1)-x(n - 1) * y(n - 2))+(x(n-1) * y0 - x0 * y(n-1)) ]/2.0

接下来就是喜闻乐见的整体代码

       import java.util.Scanner;
           
       public class Main {
            public static void main(String[] args) {
                Scanner in = new Scanner(System.in);
                while (in.hasNext()) {
                    int n = in.nextInt();
                    if (n == 0) {
                        break;
            }
            int  [] a = new int [n];
            int  [] b = new int [n];
            for (int i = 0; i < n; i++) {
                a[i] = in.nextInt();
                b[i] = in.nextInt();
            }
            double s = 0;
            for (int i = 0; i < n; i++) {
                int t = i + 1;
                if (t == n) {
                    t = 0;
                }
                s = s+(a[i] * b[t] - a[t] * b[i])/2.0;
            }
            System.out.println(Math.abs(s));
        }//s
    }
}





hdoj2045动态规划

题目链接：

【动态规划】:把多阶段过程转化为一系列单阶段问题，利用各阶段之间的关系，
逐个求解，创立了解决这类过程优化问题的新方法——动态规划。

  • 思路：设f(n)代表当有n个方格时，满足条件的涂法

    易知：f(1)=3;f(2)=6; f(3)=6;

    • 假设第n-1个格子的颜色跟第1个相同，
      因为第n-1个格子颜色已确定，第n个格子可以涂的颜色有两种，
      故为2*f(n-2)

    ​ 怎么理解: 以n=4时为例

    • R G R (P/G) 下划线两个，代表f(2),因为在第1个和第3个相同，所以相当于第3个也已经确定了 。所以是f(2)*2
    • R G P (G) 下划线三个，代表f(3),因为第1个和第3个不同，留个第4位的只有一个选择了，所以是f(3)*1

  • 答案：

​ 

import java.util.Scanner;
public class Main {
    public static void main(String[] args) {
        long [] a =new long [51];
        a[1]=3;
        a[2]=6;
        a[3]=6;
        for(int i=4;i<=50;i++)
        {
            a[i]=a[i-1]+2*a[i-2];
        }
       Scanner in  = new Scanner(System.in);
        while(in.hasNext()) {
           int n=in.nextInt();
            System.out.println(a[n]);
        }
        }
    }





进制转换加运算

import java.util.*;
public class Main {

    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        while (in.hasNext()){
            String num_01 = in.next();
            String num_02 = in.next();
            //1.先将字符串转为十六进制,十六进制转为十进制
            long result = Long.parseLong(num_01, 16)+ Long.parseLong(num_02,16);
            if(result>=0){
                System.out.println(Long.toHexString(result).toUpperCase());
            }else {
                System.out.println("-"+(Long.toHexString(Math.abs(result))).toUpperCase());
            }
        }
    }
}
